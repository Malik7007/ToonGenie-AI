## 🌟 The Studio Vision

**ToonGenie AI Studio** is a comprehensive production suite designed to bridge the gap between AI generation and professional storytelling. While most AI tools generate isolated images, ToonGenie provides a **Multi-Phase Production Pipeline** that prioritizes narrative flow and visual consistency across entire episodes.

Whether you're creating educational content, whimsical children's stories, or cinematic shorts, ToonGenie gives you director-level control over every frame.

---

## 🚀 Professional-Grade Features

### 🧠 Phase 1: AI Screenwriting & Logic
Unlike simple prompting, our **Script Editor** allows you to refine the AI's logic.
- **Story Tone Orchestration:** Choose from *Whimsical, Spooky, Educational, Epic, or Noir*.
- **Multi-Lingual Support:** Generate scripts and voiceovers in dozens of languages instantly.
- **Cinematography Directives:** Manual control over camera angles (Wide, Close-up, Bird's Eye, etc.) per scene.

### 🎭 Phase 2: Character Visual DNA Engine
We solved the "Character Consistency" problem with our **DNA Mapping** system.
- **Trait Definition:** AI assigns specific personality and visual traits to characters.
- **DNA Re-rolling:** Preview and regenerate characters until they match your vision, locking their seed and visual prompts for the entire production.
- **Master Turnarounds:** Generates high-fidelity turnaround sheets to maintain character IP across scenes.

### 🎥 Phase 3: High-Octane Rendering Pipeline
- **Veo Motion Synthesis:** Transforms static keyframes into fluid 720p/1080p animation.
- **Emotional Voiceover:** Character-specific TTS that matches the dialogue context.
- **Thematic Compositing:** Automated background music composition synced to the story's mood.

### 🍱 Multi-Model Orchestration Matrix
Switch between the world's most powerful AI brains via a single HUD:
- **Google Gemini 1.5 Pro & 2.0 Flash** (Logic & Vision)
- **OpenAI GPT-4o** (Creative Writing)
- **Groq Cloud** (Llama 3/Mixtral for ultra-fast processing)
- **Ollama (Localhost)** (Private/Offline model synchronization)

## 🎬 The Production Workflow: Step-by-Step

### Step 1: Creative Spark
Enter a descriptive prompt like *"A clumsy robot trying to bake a chocolate cake in a futuristic kitchen."* Select your target age group (e.g., Kids), your animation style (e.g., 3D Render), and the story tone (e.g., Whimsical). Click **Initiate Studio**.

### Step 2: Script Directing
The AI generates a 3-5 scene script. In this phase, you are the **Lead Writer**:
- Edit scene descriptions for better visual context.
- Adjust dialogue to match your character's voice.
- Set the **Camera Angles** for cinematic impact.

### Step 3: DNA Synthesis
The AI designs your characters. This is where you lock in **Visual Consistency**:
- Review the generated character sheets.
- Use the **Re-roll** button if a character doesn't look quite right.
- Once you approve, their "Visual DNA" is locked, ensuring they look identical in every scene.

### Step 4: Full Production & Screening
Initiate the full render. The system will:
1. Compose custom background music based on the story's mood.
2. Paint all scene keyframes using locked DNA.
3. Synthesize voiceovers with emotional depth.
4. Render motion video for every scene.

Once complete, enter **Theater Mode** to screen your masterpiece with cinematic overlays and subtitles.

## 📦 Detailed Installation

### 1. Environment Requirements
- **Node.js:** v18.17.0 or higher
- **NPM:** v9.x or higher
- **Browser:** Modern Chrome/Edge/Arc (Requires WebP and AudioContext support)

### 2. Quick Setup
```bash
# Clone the repository
git clone https://github.com/Malik7007/ToonGenie-AI.git

# Enter the directory
cd ToonGenie-AI

# Install dependencies
npm install
```

### 3. Service Configuration
Create a `.env.local` in the root directory:
```env
VITE_GEMINI_API_KEY=your_google_key
VITE_OPENAI_API_KEY=your_openai_key
# Localhost Ollama defaults to http://localhost:11434
```

### 4. Directorial Dev Mode
```bash
npm run dev
```
Open `http://localhost:3000` to enter the Studio.
